Work

The Senate Scorecard

The Senate Scorecard is a MEAN stack app that updates the the traditional non-profit Congressional scorecard. This app pulls Senate data from the Leage of Conservations Voters 2015 scorecard. It's ultimately intended to be a hub for multiple non-profit scorecards and serve as a resource for voters.

State of Choice

The State of Choice is an Angular on Rails web app utilizing reports from the Guttmacher Institute to build the API backend. The site highlights state laws that have passed since the 2010 Tea Party wave election, most of which limit access to abortion care. A user stories section also allows for a continuing dialog to help bring abortion out of the shadows.

Shit Politicians Say

Shit Politicians Say is a Ruby on Rails application that uses PostgreSQL as the database. The app aims to gamifying learning about America's politicians through a simple quiz. It also allows site visitors to upvote or downvote direct quotes from politicians.

About
